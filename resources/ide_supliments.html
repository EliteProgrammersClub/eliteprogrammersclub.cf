Web Development

W3Schools

W3Schools offers free HTML, XHTML, CSS, JavaScript, XML, ASP, PHP, SQL tutorials with lots of working examples and source code.

Emmet

Emmet offers a handy set of tools for high speed HTML and CSS coding. This tutorial here should get you started. Emmet is fully supported by Apanta studio and Bulefish text editor

Bootstrap

If you are a web programmer not so fun with web design, Bootstrap by twitter is the tool you need to democratize the whole process of web design. Go get Bootstrap and check out this examples.

Modernizr

Modernizr is a small JavaScript library that detects the availability of native implementations for next-generation web technologies, i.e. HTML5 and CSS3 specifications.

Zurb Foundation

With foundation it's now very easy to start a web project so that you can help take over the world one responsive site at a time. This should get you started

C Programming

GTK+ IDE

Bulefish text editor offers tools for high speed GTK+ programming with features like auto completion of the very long gtk+ functions and much more...

Glade

Glade is a very easy go use gui builder for gtk+ - Interface building has never been simpler

C algorithms

This is a collection of common Computer Science algorithms which may be used in C projects.

Project Docummentation

Doxygen

Doxygen is a documentation system for C++, C, Java, Objective-C, IDL (Corba and Microsoft flavors) and to some extent PHP, C#, and D.
